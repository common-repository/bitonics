== Description ==
The iconic font and CSS toolkit for cryptocurrencies
Check the Bitonics website for more information: https://bitonics.net/

Usage examples:
[bitonics currency='XLM'] 1,234
[bitonics currency='BTC'] 1,234
[bitonics currency='IOTA' showcode='0'] 1,234
...
[bitonics currency='BTC'] Bitcoin
[bitonics currency='ETH'] Ethereum
[bitonics currency='BCC'] Bitcoin Cash
[bitonics currency='XLM'] Stellar
...
[bitonics currency='MIOTA'] IOTA
[bitonics currency='DASH'] Dash
[bitonics currency='XMR'] Monero
[bitonics currency='LSK'] Lisk
...

== Installation ==
Install Bitonics using the built-in plugin installer:

1. Go to Plugins > Add New.

2. Type in the name of the WordPress Plugin (Bitonics) or descriptive keyword, author, or tag in Search Plugins box or click a tag link below the screen.

3. Find the Bitonics Plugin you wish to install.

4. Click Details for more information about the Plugin and instructions you may wish to print or save to help setup the Plugin.

5. Click Install Now to install the Bitonics Plugin.

6. The resulting installation screen will list the installation as successful or note any problems during the install.

7. If successful, click Activate Plugin to activate it, or Return to Plugin Installer for further actions.
